# Rock Paper Scissors (Python)

A simple command-line Rock–Paper–Scissors game built with Python.  
This project focuses on clean logic, input validation, and readable code.

## Features
- User vs computer gameplay
- Random computer choice
- Input validation
- Win, loss, and tie handling
- Score tracking

## How to Run
1. Make sure Python is installed.
2. Clone the repository or download the file.
3. Run the game:

```bash
python rock_paper_scissors.py
